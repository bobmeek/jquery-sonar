## jQuery sonar effect plugin

### Usage

```html
<link href="../sonar.css" rel="stylesheet">
<script src="http://ajax.googleapis.com/ajax/libs/jquery/1.10.1/jquery.min.js"></script>
<script src="../sonar.js"></script>
```

```js
$('.icon').sonar();
```
